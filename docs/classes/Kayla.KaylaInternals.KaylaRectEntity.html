    Class KaylaRectEntityInternal

    The concrete LEA entity instance created for each Kayla component/fiber.

    Wraps a LEA.RectLeaEntity and delegates update() and draw() to the component's useTick / usePaint hooks.

    Users should almost never interact with this class directly â€” use KaylaInternalRect or () / (self) instead.
